A plataforma VIDA SEGURA é uma aplicação web para visualização, análise e disponibilização de dados de acidente de trânsito consolidados anualmente na cidade de São Paulo, com interfaces gráficas, mapa, lista de registros e exportação de conjuntos de dados via arquivo csv.

PAINEL

O Painel é a interface inicial da Plataforma, com um panorama sobre os dados de acidente de trânsito em 2017, 2016 e 2015 a partir das seguintes janelas:

  1. Um mapa de pontos críticos da cidade, calculados a partir dos dados presentes na plataforma
  2. Um mapa de incidentes no último ano consolidado pela CET (2017)
  3. O cálculo dos custos dos acidentes de trânsito ocorridos no período, a partir de metodologia do IPEA com atualização de valores para 2017 pelo IPCA.
  4. O número total de acidentes ocorrido em cada ano da amostra
  5. Um gráfico exibindo a concentração dos acidentes de trânsito em termos de hora do dia e dia da semana para os 3 anos da amostra
  6. Uma lista das filtragens salvas pelo usuário, para organização das consultas específicas à base.

Sem nenhum filtro acionado, os dados mostrados correspondem ao município de São Paulo.

APLICANDO FILTROS NO PAINEL

As informações exibidas no Painel podem ser filtradas de acordo com Distritos e Prefeituras Regionais; para tanto, basta clicar em “Região” na barra superior, para selecionar se a filtragem será feita por Distrito ou Prefeitura Regional. O botão “Todo” - ao lado de “Região” - é uma lista que permite a seleção de uma das áreas selecionadas da cidade.

ACESSANDO OS DADOS

Do Painel, o usuário pode acessar e trabalhar com os dados utilizando o Mapa ou a Lista de Registros

O Mapa permite que o usuário visualize os dados de acidente de trânsito, filtre a partir das categorias no menu superior, selecione áreas específicas a partir de ferramentas de desenho no canto superior esquerdo, e mais. A Lista de Registros permite os mesmos tipos de filtragem do mapa.

Tanto o Mapa quanto a Lista de Registros podem ser acessados pelo botão Painel, que abre uma lista com as outras opções de visualizaçõo.

MAPA

Após acessar o mapa, é possível visualizar os detalhes do acidente clicando no respectivo ponto azul, e a seguir clicando em “Exibir” no pop-up que aparece na tela.

Também é possível adicionar layers ao mapa clicando no botão localizado no canto superior direito; como pontos críticos e um mapa de calor.

Na Interface de Mapa, também é possível exportar os dados em formato .csv

Ferramentas de desenho para uma seleção de área específica estão disponíveis abaixo dos botões de zoom do mapa, no canto superior esquerdo; é possível desenhar um retângulo ou outro polígono. Em ambos os casos, o mapa utilizará o polígono desenhado como área de seleção para os acidentes de trânsito, e basta clicar novamente no ícone do respectivo polígono para apagar o desenho.

GRÁFICOS

No canto inferior esquerdo, é possível acessar alguns gráficos sobre a amostra exibida; um deles é a concentração de acidentes por hora do dia e dia da semana, e o outro é o número de acidentes agregado por semana do ano.
